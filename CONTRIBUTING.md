# Contributing

This repo is a portfolio project. If you want to try it out or tweak it:

- Open an issue for bugs or ideas.
- Fork, branch, and send a pull request for code or doc changes.
- Keep changes focused; extra config or small fixes are welcome.

No formal style guide beyond what’s already in the codebase (e.g. existing naming and structure in `pipeline_config.py` and the DLT notebooks).
